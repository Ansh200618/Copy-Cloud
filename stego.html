How It Works

Think of it like writing a secret letter in invisible ink — only someone who knows the trick (and optionally, your password) can read it.

  1. 1 Hide a message inside text

    📝 You type a secret message and pick a cover emoji or word (like 📋).

    🔒 Optionally add a password — this scrambles the message so only someone with the same password can unlock it.

    ✨ We attach your message to the cover text using invisible characters — they look like nothing at all, so the result just looks like a normal emoji.

    📋 Share it anywhere — chat, email, social media. Only this tool can reveal what's hidden inside.

  2. 2 Hide a message inside a picture

    🖼️ You upload any image and type your secret message.

    🔍 We make tiny changes to the image's pixels — so small that no human eye can tell the difference.

    💾 Download the new image and share it. It looks completely normal, but your message is baked in.

🛡️

100% private — everything happens right here in your browser. No data is ever sent to any server. Your secrets never leave your device.
