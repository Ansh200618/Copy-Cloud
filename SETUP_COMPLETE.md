## 🔧 Technical Details (If You're Curious)

### How It Works:
1. **GitHub Actions Workflow**: `.github/workflows/android-build.yml`
2. **Trigger**: Automatically runs when code is pushed to main branch
3. **Build Tool**: Gradle (native Android build)
4. **Duration**: ~5 minutes
5. **Output**: APK file uploaded to Releases

### Build Process:
```
Push to main → GitHub Actions starts → Gradle builds APK → 
Upload to Releases → You download → Install on Android
```

### No External Dependencies:
- Uses Gradle (built into Android project)
- No Expo or external services
- No API tokens required
